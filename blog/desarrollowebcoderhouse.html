Desarrollo web CoderHouse, que aprendí?

Hace un par de meses cuando decidí comenzar mi carrera como desarrollador me encuentro dando el primer paso de mi ruta y comenzado por desarrollo web dentro de CoderHouse, esta nombrada “casa de programadores”.

Aprendiendo las bases para el desarrollo web de como lo es crear sitios básicos desde la parte del usuario o de tipo “frontend” como le decimos, aplicando y combinando todo lo siguiente:

- HTML
- CSS (agregando grid a diferencia de freecodecamp, hablaremos de esto en otro post)
- SASS
- Bootstrap
- Git y GitHub
- SEO
- Servidor para el sitio

Creando como proyecto final un sitio web que debía contener todos los temas mencionados más algún que otro detalle para cumplir con las normas de las entregas, como por ejemplo que deba tener al menos 5 páginas, una de ellas aplicando grid y así con cada una de ellas… algunas a mi parecer sin sentido para lo que es el sitio en sí pero no si lo pensamos en aprender y aplicar aquellos conceptos que fuimos aprendiendo.

En mi caso el siguiente sitio como entrega final, no es de mis preferidos o en el modo que a mi me gustaría crear una web para este estilo de “cliente” (en este caso la banda), pero por cuestiones más de las entregas fue tomando esa forma. Si quieres ver más de mis sitios que voy a ir creando a lo largo de mi carrera puedes acceder aquí a “.index’s in one place” (actualmente mas de 25 sitios).
