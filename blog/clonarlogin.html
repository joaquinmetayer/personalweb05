Como clonar login Facebook y GitHub (principiante)

Que es un clon? No es más que una copia idéntica de una web en donde los programadores realizamos este ejercicio a modo de prácticas, ya que esto conlleva primero que nada conocimientos sobre aquello que queremos clonar, como lo básico de HTML y CSS o si ya queremos hacer un clon más completo de alguna web más compleja y con algunas funciones, conocimiento de cómo se crean estas mencionadas. Metiéndonos a la web donde se encuentra nuestro objetivo y comenzando a inspeccionar para poder traducirla y llevarla a nuestro código y lograr el mayor acercamiento a la original.

En mi caso hice dos clones utilizando solo HTML y CSS de Facebook y GitHub como ves abajo, accede a cada una de ellas haciendo click sobre la imagen.
