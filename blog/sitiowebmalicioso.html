"Sitio web malicioso" en tu web clon? Resuelto

Si llegaste a este post probablemente creaste un clon de un sitio web como muchos desarrolladores hacen, para mostrar que podemos copiar los estilos de un sitio, hacemos estas practicas creando y copiando exactamente la misma web que elegimos, eso ya puede ser Facebook como lo es mi caso, Netflix, Instagram, Amazon, etcetera.

Al momento de deployar nuestro sitio no sucede nada hasta unos días después que queremos ingresar y nos encontramos con este cartel…

Como lo resolvemos?

Lo primero es darle a detalles dentro de esta pantalla y a reportar problema de detección donde se los redireccionará a este sitio de google donde colocaremos el link de la web con la que tenemos el problema:

Le damos a Submit Report y a esperar! Dentro de unos dias si google considera que no es un sitio maliciosodespues del reporte no saltara mas el cartel.

Ahora si podemos ingresar!

Espero que te haya sido de ayuda, nos vemos en sitios no clon!
